# 🚀 Resume Matching AI

An AI-powered Resume Analyzer that compares resumes with job descriptions using NLP and Machine Learning.

## 🔥 Features
- Upload multiple PDF resumes
- NLP-based skill extraction
- TF-IDF + Cosine Similarity scoring
- Ranking system for candidates

## 🛠️ Tech Stack
- Python
- NLTK
- Scikit-learn
- PDFMiner
- ipywidgets

## ⚙️ Setup

```bash
pip install -r requirements.txt
python -c "import nltk; nltk.download('stopwords')"
jupyter notebook
```

Open app.ipynb
